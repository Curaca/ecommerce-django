# 🛒 E-Commerce Django (Proyecto Final)

Este es el proyecto final del e-commerce desarrollado en **Django**, integrando autenticación, catálogo de productos, carrito persistente y confirmación de compra.  
Listo para uso local y para mostrar en portafolio o GitHub profesional.

---

## 📋 Características principales

✔ Autenticación de usuarios (login / logout)  
✔ Gestión de productos para administradores  
✔ Catálogo de productos persistente con Django ORM  
✔ Carrito persistente por usuario  
✔ Confirmación de compra (generación de órdenes)  
✔ Mensajes e interfaz básica con Bootstrap  
✔ Estructura modular y limpia

---

## 🧰 Tecnologías utilizadas

- Python 3.x  
- Django 6.0.1  
- SQLite (base de datos local)  
- Bootstrap (para estilos básicos)
